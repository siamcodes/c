ตัวแปร และชนิดของข้อมูล
  ตัวแปร (Variable) คือ ชื่อที่เราตั้งขึ้น เพื่อเก็บข้อมูล แล้วอ้างอิงชื่อตัวแปรแทนข้อมูล
  มันจะไปจองพื้นที่หน่วยความจำ (RAM)
ชนิดของข้อมูล (Data Type) คือชนิดของข้อมูลประเภทต่างๆ เช่น ตัวเลข (integer),
ตัวอักขระ (character), ทศนิยม (Float), ข้อความ (string), 
บูลีน (boolean), blob วัตถุ

วิธีประกาศตัวแปร
  int x;   //ประกาศตัวแปรชื่อว่า x เป็นชนิดข้อมูล integer
  int x=10; //ประกาศตัวแปร x กำหนดค่าเป็น 10
  float y; //ประกาศตัวแปร y เป็นทศนิยม
  y = 2.59; //เอาตัวแปร y เก็บค่า 2.59
  char sex ='m'; //ประกาศตัวแปร sex เก็บค่าอักขระ m (เก็บได้อักขระ)
                 ต้องเขียนภายใต้ ' ' ถ้าไม่ระบุมันจะมองเป็นตัวเลข รหัส ASCII
       Unicode เป็นรหัสที่ใช้ในคอมพิวเตอร์ ที่รองรับหลายภาษา

  char name[30]="ข้อความ";  //ประกาศตัวแปร name เป็น Array 
                 ขนาด 30 ตัวอักขระ ข้อความที่ระบุจะอยู่ภายใต้เครื่องหมาย " "

รูปแบบการแสดงผลจาก ตัวแปร
%s = string ข้อความ
%d = decimal ตัวเลขจำนวนเต็ม - , 0 , +
%f = float ทศนิยม
%c = character อักขระ

;  semi-colon
:  colon
' ' Single Quote
" " Double Quote
