ขั้นตอนการพัฒนาโปรแกรมภาษา C
1. เขียนโปรแกรม (Source Code)
- ใช้ Editor เขียนภาษาคอมพิวเตอร์ 
2. แปลภาษา คอมไพล์โปรแกรม (Compile)
- นำ Source Code มาทำการแปลเพื่อแปลจากภาษา C
 ที่มนุษย์เข้าใจเป็นภาษาเครื่อง (Binary)
------------------------------------------
  Error มี 2 ชนิด
  1.Systax Error คือเขียนไวยกรณ์ โครงสร้างภาษาผิด  
  2.Logic Error คือผิดทางตรรกศาสตร์ ใส่สูตรหรือเงื่อนไขผิด  
-------------------------------------------
การแปลภาษา (Compile) มี 2แบบ
1. การแปลแบบ Compiler คือแปลทั้งหมด ทั้งปรโยค
ข้อดี		ข้อเสีย
- แปลได้รวดเร็ว	-ตรวจสอบข้อผิดพลาดได้ยาก
2. การแปลแบบ Interpreter คือแปลทีละบรรทัด
ข้อดี		ข้อเสีย
-หาข้อผิดพลาดได้ง่าย	-ช้า...ต้องแปลทีละบรรทัด
--------------------------------------------
3.การเชื่อมโยงโปรแกรม (Link)
-จะเอา Source Code มาเชื่อมกับ Library ก่อนทำให้ได้ไฟล์ 
Executable Program หรือที่เรียกว่า .exe ที่สามารถจำไปใช้ได้
--------------------------------------------
4. การประมวลผล (RUN)
-นำไฟล์ .exe มาแสดงผล (Output)
Hard Copy : อุปกรณ์ที่แสดงผลแล้วไม่หายไป เช่น Printer
Soft Copy : อุปกรณ์แสดงผลชั่วคราว เช่น Monitor , Speaker
--------------------------------------------
